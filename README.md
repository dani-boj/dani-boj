## 🌱 About me
While my background includes a degree as a Software Development Technician, I currently work as a System Administrator. My journey started in web development, which led me to study software design and development, but my curiosity and drive for continuous improvement ultimately guided me toward the world of system administration.

Today, as **Systems Director and Security Lead at Webdreams**, I oversee server environments, networks, firewalls, cloud services, and containerized platforms. 

My mission is to bridge the gap between development and operations, leveraging my Python scripting knowledge to automate and secure environments.

**Resourceful, committed, and continuously evolving.**
